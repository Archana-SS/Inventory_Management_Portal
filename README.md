# Inventory Management Portal

A web-based inventory management system built with **FastAPI** (backend) and **React + Vite** (frontend), using **SQLite** as the database.  
This project allows you to **add, update, delete, and view products** in your inventory.

---

## Features

- Add new products with name, description, price, and quantity
- Update existing product information
- Delete products
- View all products or individual products
- Built with FastAPI (Python) backend and React + Vite frontend
- Uses SQLite for local development (no external DB setup required)

---

## Tech Stack

- **Backend:** Python, FastAPI, SQLAlchemy, SQLite  
- **Frontend:** React, Vite, JavaScript  
- **Database:** SQLite (local, no installation needed)  
- **API Documentation:** FastAPI Swagger UI (`/docs`)
